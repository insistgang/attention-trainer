# 专注力训练

一个用于提升专注力的 React Web 应用，包含三种科学的注意力训练方法。

## 🎮 训练项目

### 1. 舒尔特方格 (Schulte Grid)
按 1→25 顺序快速点击数字，训练视觉注意力广度。

### 2. N-Back
判断当前字母是否与 N 步前相同，训练工作记忆。

### 3. Stroop 测试
说出文字的颜色而非文字内容，训练抗干扰能力。

```bash
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 📝 使用建议

- 每天训练 10 分钟
- 轮流做 2-3 种练习，每种 3-5 轮
- 推荐顺序：舒尔特方格热身 → N-Back 核心训练 → Stroop 收尾
